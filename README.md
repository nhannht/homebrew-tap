# nhannht/homebrew-tap

Homebrew tap for [nhannht](https://github.com/nhannht)'s macOS apps.

## Usage

```sh
brew install --cask nhannht/tap/ncomphunt
```

Or tap first, then install by short name:

```sh
brew tap nhannht/tap
brew install --cask ncomphunt
```

## Casks

- [ncomphunt](https://github.com/nhannht/ncomphunt) - native macOS app that finds
  and indexes competitions (competitive programming, AI challenges, CTFs,
  hackathons, design contests), Vietnam-first and global.
